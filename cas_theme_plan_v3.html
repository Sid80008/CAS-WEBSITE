Central Academy Anta · Theme Plan v3 · June 2026

Not a Website.
An Experience.

The previous version rearranged the furniture. This one burns the house down and builds something that makes parents feel something — curiosity, pride, trust. The kind of site they send to their relatives.

Cinematic. Editorial. Immersive. Distinctly Indian.

Core Design Philosophy

Scroll through a school.
Not a webpage.

Every section is a scene. Every transition is a camera move. The user doesn't read about Central Academy — they walk through it. Like the first day of school.

Direction Cinematic Storytelling
Palette Midnight Saffron
Typography Playfair Display + Plus Jakarta Sans
Signature Diagonal Transitions
Scroll Feel Scene → Scene → Scene
What changed from v2: v2 was a polished version of the old site — same blue/amber, same horizontal sections, same "professional school website" feeling. v3 is a completely different philosophy. We're stealing the "world building" and "cinematic scroll" from the inspiration images, adapted for a school context. The colour is new. The typography is new. The section transitions are new. The hero philosophy is new.

Chapter 06

Motion System

Motion creates the cinematic feel. Every animation has one job. Nothing moves without a reason. Speed = confidence.

Page Load — Hero Reveal
opacity: 0 → 1 (1.8s)
scale: 1.06 → 1 (1.8s)
ease: [0.22, 1, 0.36, 1]
The campus photo zooms out as it fades in. Like a camera pulling back to reveal a world. The single most important animation on the site.
Text Stagger — Hero
eyebrow: delay 0.4s
h1: delay 0.65s (y: +36px)
sub: delay 0.85s
btns: delay 1.05s
Text reveals one by one from bottom. The headline arrives last — creates anticipation. Stolen from film title reveals.
Scroll — Section Reveal
y: 48px → 0
opacity: 0 → 1
duration: 0.85s
stagger: 0.1s per child
Cards in grids reveal one at a time as you scroll. Not all at once — that's lazy and fails to create momentum.
Hero Parallax
background-position: 0 → -15% on scroll
(desktop only)
Photo moves at 0.15x scroll speed. The depth layers shift at different rates — creates the 3D world feeling from Kamui/Soul.
Stat Counter
0 → target value
duration: 2.2s
ease: easeOut cubic
trigger: IntersectionObserver
Fires once when stat bar enters viewport. 30+ counts up from 0. Emotional. Proves the school is real and has history.
Card Hover
translateY: -6px
shadow: 0→16px
border-color: → saffron
duration: 220ms
Cards lift when hovered. Saffron border glow appears. Confirms the element is interactive without being jarring.
Gallery Hover
scale: 1.05
overlay: opacity 0 → 0.35
overlay color: saffron
duration: 350ms
Photos breathe on hover. Saffron tint overlay reinforces brand. Eye icon fades in at center.
Notice Ticker
marquee: CSS animation
duration: 45s
direction: left
pause on hover
Scrolling notice strip below the stat bar. Fast enough to be dynamic, slow enough to be readable. Pauses when mouse enters.
Portal — Tab Switch
opacity: 0 → 1 (200ms)
x: -12px → 0
ease: easeOut
Portal content slides in when switching tabs. Directional: forward tabs slide right→left, back tabs slide left→right. Users feel navigation momentum.
Accessibility rule: @media (prefers-reduced-motion: reduce) disables all transforms and parallax. Counter becomes instant. Page fades remain (functional, not decorative). This is already in globals.css.

Chapter 08

The Laws

Eight non-negotiable design laws. If any component breaks any of these, it gets rebuilt.

01
Depth Over Flatness
  • Three layers in every hero: bg photo / directional gradient / text
  • Cards overlap section boundaries — never isolated in their own box
  • Quick-access cards start inside hero and extend below stat bar
  • Photo sections use the subject as a foreground element, not a background fill
02
Scenes, Not Sections
  • Every section has a "scene name": Opening Shot, The Lobby, The Toppers Reveal
  • Diagonal clip-path transitions between sections — never flat horizontal borders
  • Background alternates: dark → light → saffron tint → dark — creates rhythm
  • Each section should feel like entering a different room of the same school
03
Saffron is Sacred
  • Saffron (#E8621A) appears in gradient form on primary CTAs, stat bar border, topper rank badges
  • Never as a large fill — always as a highlight, glow, or accent line
  • Against navy: it glows. Against white: it anchors. Against cream: it warms.
  • The old amber #fdad4e is fully retired. It appears nowhere in the new system.
04
Editorial Typography
  • Playfair Display for hero/display titles ONLY — creates serif editorial gravitas
  • Italic emphasis in Playfair for the "emotional word" in a headline
  • Oversized watermark text (5–8% opacity) as section background element
  • Never two display font families on same page — Playfair + Jakarta only
05
Real Over Perfect
  • Real school photos even if slightly imperfect — AI stock destroys trust instantly with Indian parents
  • Real topper names in the hero, not generic "Award Winner" placeholder
  • Principal signature scan, not typed name, in the message
  • Staff initials avatars better than "photo loading" broken images
06
Hindi is First Class
  • Every notice: English title + Hindi subtitle at equal visual weight
  • Language toggle: prominent in nav, never hidden or tiny
  • Homepage CTA: "Apply Now / अभी आवेदन करें" — both visible
  • Font system: Noto Sans Devanagari as fallback, no flash-of-wrong-font
07
Mobile is Primary
  • All touch targets: 44×44px minimum. This is a Rajasthan school — mobile-first audience.
  • Hero: 100svh on mobile, 680px fixed on desktop
  • Portal: bottom nav bar on mobile, sidebar on desktop
  • All content readable on 375px without horizontal scroll
08
Performance is Respect
  • All hero images: WebP format, next/image with priority=true
  • Fonts: preload only Playfair 700+900, Jakarta 700+800+900, Inter 400+600
  • Portal dashboard: <2s on 4G (Anta network speeds)
  • Never load more than 6 images above the fold
What This Is Not
✕ Not a Government Portal
No full-width grey headers. No blinking "NOTICE" text. No table-based layouts from 2008. No Visit Counter widget. No "Best viewed in Chrome" footnote.
✕ Not a SaaS Landing Page
No white cards floating on white. No feature grids with rocket icons. No "Start for Free" hero hook. No gradient mesh backgrounds. No social proof avatar clusters.
✕ Not a WordPress Theme
No colored banner every 200px. No 4 font families per page. No widget-box sidebar. No "powered by" footer spam. No generic slider with crossfade.
✕ Not an AI Template
No glassmorphism everywhere. No neon particle animations. No "smiling diverse students" stock photos. No purple-to-pink gradient. No floating blob backgrounds.

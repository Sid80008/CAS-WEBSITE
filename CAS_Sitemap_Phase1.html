API

REST API Endpoints — 16 Routes

app/api/
Endpoint Methods Purpose Auth
/api/auth/[...nextauth] GET POST NextAuth.js session handler — login, logout, session refresh —
/api/notices GET Paginated public notices, pinned first, then by date. Query: page, limit, lang Public
/api/gallery GET Album list with cover image URL + photo count. Query: category Public
/api/gallery/[id] GET All images in specific album with Cloudinary URLs + captions Public
/api/staff GET All 37 staff members ordered by display_order Public
/api/toppers GET Toppers filtered by session + class query params Public
/api/tc-verify POST Search TC records by name + roll + year. Returns TC record or not-found. Public
/api/contact POST Contact form submission → sends email via Resend to school email Public
/api/admission-enquiry POST Save enquiry to AdmissionEnquiry table + send auto-reply email via Resend Public
/api/admin/notices GET POST PUT DEL Full CRUD for notices. POST triggers revalidatePath('/notices') for instant ISR refresh. ADMIN
/api/admin/gallery GET POST PUT DEL Album + image CRUD. Handles Cloudinary upload confirmation and reordering. ADMIN
/api/admin/staff GET POST PUT DEL Staff CRUD — create, update photo/designation/order, delete ADMIN
/api/admin/upload POST Generates signed Cloudinary upload URL for direct browser-to-CDN uploads ADMIN
/api/payment/create-order POST Creates Razorpay order via SDK. Returns order_id for checkout widget. PARENT
/api/payment/verify POST Verifies Razorpay signature with crypto.createHmac. Updates FeeRecord on success. PARENT
/api/revalidate POST On-demand ISR revalidation for any path. Called internally by admin mutations. ADMIN
